## Автоматизированная информационная система “Учёта и контроля инвентаризации”.

## 1. Описание проекта
 Приложение для автоматизации работы службы такси.

---

## 2. Цель и задачи проекта
**Цель:** Автоматизация операционных процессов службы такси.

**Задачи:**
- Управление поездками
- Динамический расчёт стоимости
- Управление водителями
- Финансовые операции

---

## 3. Функциональность
- Ввод адреса
- Выбор тарифа
- История поездок
- Список водителей
- Отзывы
- Оплата

---

## 4. Архитектура проекта
Тип архитектуры: Многослойная (Layered / N-Tier)

**Технологии:**
- Visual Studio 2022
- Wpf
- C#
- SQL Server Management Studio
- MSTest
- Entity Framework
